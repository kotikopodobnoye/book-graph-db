Задача:

Необходимо создать базу данных для книжной библиотеки, которая будет содержать информацию о книгах, авторах, издательствах и жанрах.

Структура базы данных:

1. Таблица узлов Authors с полями:
    AuthorID (идентификатор автора)
    FullName (полное имя автора)
    Birthday (дата рождения автора)

2. Таблица узлов Books с полями:
    BookId (идентификатор книги)
    Title (название книги)
    PublicationYear (год публикации книги)
    ISBNCode (код ISBN книги)

3. Таблица узлов Genre с полями:
    GenreId (идентификатор жанра, автоинкрементируемый)
    Title (название жанра)
    Description (описание жанра)

4. Таблица узлов Publishers с полями:
    PublisherId (идентификатор издательства, автоинкрементируемый)
    Title (название издательства)
    Location (местоположение издательства)

5. Таблица ребер AuthorOf для связи авторов с книгами

6. Таблица ребер GenreOf для связи жанров с книгами

7. Таблица ребер PublishedBy для связи издательств с книгами

Запросы:

1. Выбор автора и его книг
2. Выбор жанра и соответсующих ему книг
3. Найти все книги, написанные автором "X", опубликованные в жанре "Y"
4. Найти все книги, изданные издательством "X", опубликованные в жанре "Y"
5. Найти все книги, впервые изданные в период с 1800 по 1900 с жанром "Y"
6. Цепочка книг для "Войны и мир", у которых есть общий жанр (т.е. начиная с например Романа), до 2 элементов
7. Самый короткий путь между книгами (по жанрам)
